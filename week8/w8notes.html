CSS3 Transforms and Transitions

Transforms

What does it do?- enables translation, rotation, scale, and/or skew any element on the page.We use transform functions to manipulate the appearance

1.Translation

Translation functions allow movement of elements to the left, right, up, or down

These functions are similar to the behavior of position: relative; when declaring top and left, moving elements up and down or left and right along the x and y axis

A translated element can only be moved relative to its current position without impacting the flow of the document.

translatex or translatey move an element vertically or horizontally

e.g to move up the y-axis by 30px: transform: translateY(-30px)

Note: transforms must have a prefix -webit- to support older browser versions. Also, transforms don't work on inline elements like *span, you must change display: inline-block;

2.Scaling

The scale(x,y) function scales an element by the defined factors horizontally then verticall. If one value is provided, it will be used for both x and y values

For example- scale(1.5, 0.25 will distort the element

A scaled element will grow outwards from or shrink inwards towards its center

3.Rotation

The rotate() function rotates an element around the pointof origin a specified angle value.As with scale, by default the point of origin is the element's center

values must be in degrees:

    transform: rotate(10deg) translateX(40px) scale(1.5);
    The example above rotates the element by 10 degs clockwise, moving it along the x-axis by 40px and making it larger by 1.5
    Note: transforms are applied in the order provided
