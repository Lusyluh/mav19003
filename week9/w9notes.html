The window object

The window object represents an open window/page in a browser.

The Browser Object Model (BOM)

This is a collection of properties and methods that contain information about the browser and computer screen. For example, we can find out which browser is being used to view a page, find out the dimensions of the screen it is viewed on, and which pages have been visited before the current page.

Dialogs : Window Object Methods

  • window.alert("message")
  • window.confirm("OK or Cancel")
  • window.prompt("Please Enter...")

Find out which browser is being used

The window object has a navigator property that returns a reference to the Navigator object. The Navigator object contains information about the browser being used.

  • window.navigator.userAgent returns: "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_3) ➥ AppleWebKit/602.4.8 (KHTML, like Gecko) Version/10.0.3 ➥ Safari/602.4.8"
  • Note:Not reliable, users can modify this info

window.location

The window.location property is an object that contains information about the URL of the current page. It contains a number of properties that provide information about different fragments of the URL

  • window.location.href : returns full URL
  • window.location.protocol : returns a string describing the protocol used(such as http, https,pop2,etc)
  • window.location.host : returns a string describing the domain of the current URL and the port number
  • window.location.hostname : returns a string describing the domain of the current URL
  • window.location.port: returns port number
  • window.location.search : returns a string that starts with a '?' followed by the query string parameters. It returns an empty string if there are no query string parameters.
  • window.location.hash :returns a string that starts with a "#" followed by the fragment identifier. It returns an empty string if there is no fragment identifier

The Browser History

window.history -- use it to access information about any previously visited pages in the current browser session

  • window.history.length : shows how many pages have been visited before arriving at the current page.
  • window.history.go() : can be used to to a specific page, 0 is the current pages
  • window.history.go(1); // goes forward 1 page
  • window.history.go(-1); // goes back 1 page
